# Project Information

As the name implies, Tridigit Calculator is a three digit calculator. It takes an equation as input and outputs the answer. An example of the equation is **420+420**. 

# Usage

In the command prompt, run **python *path/to/file/TridigitCalculator.py* "aaaxbbb"**, where aaa and bbb are the 2 three digit numbers and x is the operator.
